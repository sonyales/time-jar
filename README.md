# ⏳ TimeJar

**TimeJar** — приложение для тех, кто ждёт.

Это визуальный трекер ожидания, который помогает отслеживать прогресс до важной даты. Пользователь выбирает период ожидания, а приложение показывает, какая часть этого периода уже прошла.

Главная идея проекта — превратить ожидание во что-то наглядное, понятное и приятное. Вместо абстрактного «ещё долго» пользователь видит прогресс в виде банки, которая постепенно наполняется. Количество дней может пугать, но в процентах всё становится проще и спокойнее.

🔗 **Demo:** [time-jar-nu.vercel.app](https://time-jar-nu.vercel.app/)

---

## ✨ О проекте

**TimeJar** помогает визуализировать ожидание и сделать его менее абстрактным.

Приложение можно использовать для любого события, которого хочется дождаться:

- до отпуска
- до релиза
- до встречи
- до дедлайна
- до экзамена
- до праздника
- до поездки
- до личной цели
- до любого важного события

Пользователь задаёт дату начала и дату окончания ожидания, а приложение рассчитывает прогресс и показывает, сколько времени уже прошло.

---

## 🫙 Основная идея

Ожидание часто кажется бесконечным.  
TimeJar делает его видимым.

Вместо того чтобы просто считать дни, приложение показывает прогресс в виде банки с жидкостью. Чем ближе событие, тем больше наполняется банка.

Так ожидание превращается в понятный визуальный процесс.

---

## 🚀 Основной функционал

- выбор даты начала ожидания
- выбор даты окончания ожидания
- расчёт прогресса между двумя датами
- отображение прогресса в процентах
- визуализация прогресса в виде банки с жидкостью
- плавная анимация заполнения банки
- понятный и минималистичный интерфейс
- адаптивная логика отображения прогресса

## 📦 Установка и запуск

Склонируйте репозиторий:

```bash
git clone https://github.com/sonyales/time-jar.git
```

Перейдите в папку проекта:

```bash
cd time-jar
```

Установите зависимости:

```bash
npm install
```

Запустите проект в режиме разработки:

```bash
npm run dev
```

После запуска приложение будет доступно в браузере по адресу, который покажет Vite в терминале.
